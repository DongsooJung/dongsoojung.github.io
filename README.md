## 🎨 Design System

- **컬러**: 딥 블랙 베이스 (`#0a0a0c`) + 액센트 `#4f7df5` (blue) / `#c4a265` (gold)
- **타이포**: [Instrument Serif](https://fonts.google.com/specimen/Instrument+Serif) (heading) · [DM Sans](https://fonts.google.com/specimen/DM+Sans) (body) · JetBrains Mono (code)
- **모션**: radial pulse 애니메이션 (hero glow), 스크롤 smooth behavior
- **반응형**: 1200px max-width 그리드, 모바일 1 column 전환

## 🛠 Tech Stack

- **정적 사이트**: 순수 HTML + CSS (프레임워크 無)
- **호스팅**: GitHub Pages (user page, `main` branch 루트)
- **폰트**: Google Fonts preconnect 최적화
- **SEO**: OG tags (`og:title` / `og:description` / `og:type` / `og:url`) 완비
